# Sample Layout

```text
config/audiobridge/input/
  music/
    atlantis_intro.mp3
  ambience/
    temple_loop.mp3
  ui/
    confirm.mp3

config/audiobridge/cache/
  index.txt
  <hash>.ogg
```

Example mappings:

- `music/atlantis_intro.mp3` -> `audiobridge:music.atlantis_intro`
- `ambience/temple_loop.mp3` -> `audiobridge:ambience.temple_loop`
- `ui/confirm.mp3` -> `audiobridge:ui.confirm`
